# Spark Value

An interactive, scroll-driven presentation exploring how AI can transform marketing the way it transformed development. Built with React, Vite, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Sections

0. **Hero** — "What if AI could do for marketers what it did for developers?"
1. **Dev Transform** — How AI changed development
2. **Marketing Pain** — The fragmentation problem in marketing
3. **Funnel** — The marketing funnel, reimagined
4. **Competitors** — Competitive landscape
5. **Surface** — The unified surface
6. **Audience** — Audience insights and segmentation
7. **Collaboration** — Team collaboration workflows
8. **Brief to Campaign** — From strategy to execution
9. **Closing** — The structural advantage: context + tools + iteration
